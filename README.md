# 대학생활정보 큐레이션 서비스 슥삭 - iOS



## About

대학교 포스터 게시판은 덕지덕지, 스펙 웹사이트들은 어질어질 한 대학생들을 위해 대학생들이 직접 만들었습니다.
나만의 대학생활 캘린더 <슥삭>은 대외활동, 공모전, 인턴, 동아리, 장학금, 강연 등 대학생에게 필요한 모든 정보를 1분만에 슥삭 정리하는 대학생 필수어플 입니다.
